基于 React + Ant Design 的项目初始模板

## 模板特点

### 主流框架 & 特性

+ Ant Design Pro 6.0.0
+ React 18.2.0
+ node 至少 16 版本及以上
+ antd 5.2.2
+ Type Script
+ 动态路由
+ Eslint
+ Prettier

### Ant Design Pro 架构

#### Umi

+ Node.js 前端开发基础环境
+ webpack 前端必学必会的打包工具
+ react-router 路由库
+ proxy 反向代理工具
+ dva 轻量级的应用框架
+ fabric 严格但是不严苛的 lint 规则集（eslint、stylelint、prettier)
+ Type Script 带类型的 JavaScript
